개인정보 처리방침

본 방침은 'Klock' 앱의 데이터 처리 방식을 설명합니다.

1. 데이터 처리 원칙

  • 온디바이스 AI: 모든 인물 인식 AI 연산은 사용자의 스마트폰에서만 실시간으로 수행됩니다.
  • 서버 전송 없음: 촬영된 이미지나 영상 피드는 절대로 외부 서버로 전송되지 않습니다.
  • 얼굴 식별 불가: 당사는 사용자의 구체적인 얼굴 특징이나 생체 정보를 수집하거나 식별하지 않습니다.

2. 데이터 저장

타임랩스 영상과 공부 통계 기록은 사용자의 기기 로컬 저장소에만 안전하게 보관됩니다. 사용자가 앱을 삭제하거나 데이터를 직접 지우면 복구가 불가능합니다.
